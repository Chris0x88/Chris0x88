## How I work

I favour the **simplest architecture that ships**. Spec first, working software next, polish last. Built in the open, for real users, and measured by what they actually do with it.

**Biases:** Edge-first · Local-first · Own-your-data · Zero unnecessary dependencies · No vendor lock-in.
